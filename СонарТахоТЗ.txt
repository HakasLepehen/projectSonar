Техническое задание (то, как видится мне админка проекта)

НАВИГАЦИОННОЕ МЕНЮ. Ничего особенного, логику ее работы я уже описал в верстке.

СПИСОК АВТОМОБИЛЕЙ. Здесь должен отображаться список всех автомобилей, которые предложены к выбору 
в выпадающем списке на главной странице. При выборе одним щелчком мыши на названии конкретного автомобиля
и нажатии на кнопку "Редактировать список работ" в блоке ниже должен выводиться список работ для него. 
При двойном нажатии на названии авто - должно редактироваться его название. Список должен быть редактируемым.

кнопка ДОБАВИТЬ АВТОМОБИЛЬ - добавляет автомобиль в список. По умолчаниюназвание должно быть "Новый автомобиль".
Список работ в блоке редактирования для "Новый автомобиль" должен быть пустым, без полей для ввода и кнопок удаления.

кнопка "Удалить автомобиль" вызывает диалоговое окно в браузере для подтверждения или отмены операции
удаления автомобиля из списка. При этом, элементы из блока снизу должны скрываться или удаляться со страницы.

кнопка "Добавить вид работ" добавляет  формы для ввода с надписями "Введите наименование работ" и "Введите стоимость".
Также должна добавляться на страницу кнопка "Удалить".

кнопка "Удалить" удаляет строку с наименованием работы и стоимостью работы из списка. После удаления, 
кнопка должна исчезнуть.

кнопка "Сохранить" сохраняет внесенные изменения в базу данных.
